Getting started with Javascript

Synax

console.log() print out
confirm() to ask question
prompt to receive feedback
=== equal to
!== not equal to

Function

  • To declare a function using var.
  • The name should begin with a lowercase letter and the convention is to use lowerCamelCase.
  • Each line of code inside { } must end with a semi-colon ;.
  • The entire function ends with a semi-colon ;.

Global vs Local Variables

  • If var is used outside a function, that variable has a global scope; If var is used inside a function, that variable has a local scope.
  • Different form Python, if you want to assign a global variable, you don’t need to use golbal in front of the global variable.

Control Flow

  • swith(expression) { case ' ' }
  • logical operators: and (&&), or (||), and not (!).
  • isNaN() checks to see if that thing is not a number.
    Be careful: if you call isNaN on a string that looks like a number, like ‘42’, JavaScript will try to help by automatically converting the string ‘42’ to the number 42 and return false (since 42 is a number).

Arrays

  • Heterogeneous arrays: var mix = [42, true, "towel"];
  • Multidimensional arrays: N*N;
  • Jagged arrays: not N*N;

Objects

Objects are just collections of information (keys and values) between curly braces.

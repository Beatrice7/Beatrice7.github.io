Hexo出现Error deployer not found:github 的错误

好长时间以来写的东西都零零散散，一直没有发布博客。搞好了一篇像样点的想发布吧，hexo还出了点小问题。

Error deployer not found:github

出现上述问题的原因很简单就是hexo 更新到3.0之后，deploy的type 的github需要改成git，另外还要注意yaml语法要求严格，配置那里的冒号后面是有一个半角的空格的.但问题是我改了之后仍然有问题，变为

Error deployer not found:git

查了一下得知，我们可能还需要执行一步

npm install hexo-deployer-git --save

BUT我又出现了这样的错误

zsh:command not found: npm

无奈，那就重装一下node.js和npm吧

brew install node

可是人家告诉我已经装了node- already installed, 所以我只好去到官网下载然后安装。可能是我下载的是比之前的版本新的缘故，一些module里的东东还是之前版本的，于是当我执行hexo g的时候又出现了

Error: Module version mismatch.

在stackoverflow找到的解决方法如下

rm -rf node_modules
npm update

###问题终于都解决了，我应该愉快的写博客了啊喂！

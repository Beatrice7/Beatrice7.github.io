jQuery

What is jQuery

jQuery is a library, or set of helpful add-ons, to the JavaScript programming language.

Getting Started

we’ll need to start up our jQuery magic using the $(document).ready(); syntax you’ve seen. It works like this:
– $() says, “hey, jQuery things are about to happen!”

– Putting document between the parentheses tells us that we’re about to work our magic on the HTML document itself.

– .ready(); is a function, or basic action, in jQuery. It says “hey, I’m going to do stuff as soon as the HTML document is ready!”
– Whatever goes in .ready()‘s parentheses is the jQuery event that occurs as soon as the HTML document is ready.
So, 

$(document).ready(something);

says: “when the HTML document is ready, do something!”

Note that .ready(); ends with a semicolon;. This tells jQuery that you’re done giving it a command.

Remember, when we say “function,” “action.” Functions are the basic unit of doing work in jQuery.

A function(){} inside the .ready().

jQuery Functions and Selectors

Functions

you can give a function just about anything as an input—even another function! That’s why .ready() can take function between its parentheses; it’s taking a function as input.

Variables

var lucky = 7;
var name = "Codecademy";
var $p = $('p'); //jQuery selector

We can put any CSS selector in quotes and pass it into $(). This means we can select classes and IDs, too!

compound selector

use comma,to affect various objects.

$('p, li').fadeTo('slow', 0);

this

Instead of using fadeOut() on all ‘div’s, we’re just using it on this div, where this refers to the div you’re clicking on.

Modifying HTML Elements

.append() will make the paragraph the last child of each div
.prepend() will make the paragraph the first child of each div.
We can specify where in the DOM(Document Object Model) we insert an element with the .before() and .after() functions.

Note: This does not copy the element from one location to another, it moves the original element effectively saving you from having to delete the original

.empty() deletes an element’s content and all its descendants. For instance, if you .empty() an 'ol', you’ll also remove all its 'li's and their text.

.remove(), not only deletes an element’s content, but deletes the element itself.

jQuery includes two functions, .addClass() and .removeClass(), that can be used to add or remove a class from an element.

Remember: You aren’t selecting anything, you are modifying your element. This means that you do not need # or . before your class.

If the element it’s called on has the class it receives as an input, .toggleClass() removes that class; if the target element doesn’t have that class, .toggleClass() adds it.

Because resizing elements is so common, jQuery has specific .height() and .width() functions that can be used to change the heights and widths of HTML elements.

jQuery also includes a general-purpose .css() function that takes two inputs: the first is the CSS element to alter, and the second is the value to set it to.

Finally, we can update the contents of our HTML elements—that is, the bit between the opening and closing tags—using the .html() and .val() functions.

$('div').html("I love jQuery!");

will set the contents of the first div it finds to “I love jQuery!”

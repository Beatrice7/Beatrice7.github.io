jQuery Events

.dblclick()—The .dblclick cause a jQuery effect when a user double clicks on an element, rather than just single-clicking.
.addClass()，.removeClass()—create an effect when your mouse is on top of an object, then have that effect vanish when your mouse moved away.

.focus()—The .focus() event handler only works on elements that can receive focus—the list of these elements is a bit vague, but HTML elements like <textarea>s and <input>s are the usual suspects.

.keydown()—The .keydown() event is triggered whenever a key on the keyboard is pressed.
.animate()—The .animate() effect takes two inputs: the animation to perform, and the time(milliseconds) in which to perform the animation.
(1,000 milliseconds = one second.)
